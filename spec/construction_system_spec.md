## 1. ОБЗОР СИСТЕМЫ

### 1.1 Концепция

Система строительства в Colony & Conquest охватывает полный цикл: от размещения **одной землянки** до планирования **мегаполиса** с инфраструктурой, зонами и многоуровневыми конструкциями.

### 1.2 Ключевые особенности

| Особенность | Описание |
|-------------|----------|
| **Эпохи** | От землянок (Эпоха 1) до небоскрёбов (Эпоха 5) |
| **Зонирование** | Жилые, промышленные, военные, сельскохозяйственные зоны |
| **Инфраструктура** | Дороги, электричество, вода, связь |
| **Многоэтажность** | Подвалы, наземные этажи, крыши |
| **Модульность** | Здания состоят из модулей, можно расширять |

### 1.3 Масштабирование

```
ЭПОХА 1 (XVI-XVII век):          ЭПОХА 5 (Будущее):
┌─────────────┐                  ┌─────────────────────────────┐
│  Землянка   │                  │  Небоскрёб 50 этажей        │
│  2x2        │                  │  20x20 база                 │
│  1 жилец    │                  │  5000 жильцов               │
│  Дерево     │                  │  Сталь, стекло, композиты   │
└─────────────┘                  └─────────────────────────────┘
```

## 5. МЕХАНИКИ СТРОИТЕЛЬСТВА

### 5.1 Этапы строительства

```
ЭТАПЫ СТРОИТЕЛЬСТВА ЗДАНИЯ:

1. ПЛАНИРОВАНИЕ (Planning)
   └─► Проверка требований (земля, ресурсы, рабочие)
   └─► Разметка территории
   └─► Время: мгновенно

2. ПОДГОТОВКА (Preparation)
   └─► Доставка материалов на площадку
   └─► Время: зависит от логистики

3. ФУНДАМЕНТ (Foundation)
   └─► Рытьё котлована (если нужно)
   └─► Заливка фундамента
   └─► Время: 10% от общего

4. КАРКАС (Framing)
   └─► Стены, перекрытия
   └─► Время: 30% от общего

5. ОТДЕЛКА (Finishing)
   └─► Крыша, окна, двери
   └─► Внутренняя отделка
   └─► Время: 40% от общего

6. ОБОРУДОВАНИЕ (Equipment)
   └─► Установка мебели, оборудования
   └─► Подключение к инфраструктуре
   └─► Время: 20% от общего

7. ГОТОВО (Completed)
   └─► Здание вводится в эксплуатацию
```

### 5.2 Формула скорости строительства

```
ВРЕМЯ_СТРОИТЕЛЬСТВА = Базовое_время × 
                     (1 / Количество_рабочих^0.7) ×
                     (1 / Средний_навык_строительства) ×
                     (1 / Качество_инструментов) ×
                     Модификатор_погоды ×
                     (1 + Штраф_ночь)

ГДЕ:
- Количество_рабочих: оптимум = 2-4 для маленьких, 10-50 для больших
- Средний_навык_строительства: 0.5 (новичок) до 2.0 (мастер)
- Качество_инструментов: 0.5 (плохие) до 1.5 (отличные)
- Модификатор_погоды: 1.0 (ясно), 1.3 (дождь), 1.5 (снег/буря)
- Штраф_ночь: +0.5 (без освещения), 0 (с освещением)
```

### 5.3 Пример расчёта

```
Строительство: Дом (Эпоха 1)
Базовое время: 4 часа = 240 минут
Рабочие: 3 человека
Средний навык: 1.2 (подмастерье)
Инструменты: 1.0 (обычные)
Погода: Ясно (1.0)
Время суток: День (0)

Время = 240 × (1 / 3^0.7) × (1 / 1.2) × 1.0 × 1.0 × 1.0
      = 240 × 0.48 × 0.83 × 1 × 1 × 1
      = 96 минут (1 час 36 минут)
```

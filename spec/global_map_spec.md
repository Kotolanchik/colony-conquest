## 1. ОБЗОР СИСТЕМЫ

### 1.1 Концепция

Глобальная карта в Colony & Conquest — это процедурно генерируемый мир с:
- **Континентами и островами** различного размера
- **Биомами** с уникальными ресурсами и климатом
- **Распределением ресурсов** стимулирующим экспансию и торговлю
- **Стратегическим уровнем** для перемещения армий и контроля территорий

### 1.2 Масштабы

| Уровень | Размер | Детализация | Использование |
|---------|--------|-------------|---------------|
| **Глобальный** | 1000x1000 км | Регионы, фракции | Стратегия, дипломатия |
| **Региональный** | 100x100 км | Биомы, ресурсы | Экспансия, колонизация |
| **Локальный** | 10x10 км | Здания, юниты | Тактика, строительство |
| **Тактический** | 1x1 км | Индивидуальные юниты | Бой |

### 1.3 Генерация мира

```
ПРОЦЕДУРНАЯ ГЕНЕРАЦИЯ:

1. SEED → Уникальный мир
   │
2. ТЕКТОНИКА → Континенты, горы, океаны
   │
3. КЛИМАТ → Температура, осадки
   │
4. БИОМЫ → Леса, пустыни, тундры
   │
5. ГИДРОЛОГИЯ → Реки, озёра
   │
6. РЕСУРСЫ → Рудные жилы, нефть, и т.д.
   │
7. ФРАКЦИИ → Начальные позиции ИИ
   │
8. ОСОБЫЕ МЕСТА → Руины, артефакты
```

## 4. ТЕРРИТОРИИ И КОНТРОЛЬ

```
ИЕРАРХИЯ ТЕРРИТОРИЙ:

ГЛОБАЛЬНЫЙ УРОВЕНЬ:
├── Континент (размер: 500x500 км)
│   └── Регион (размер: 50x50 км)
│       └── Провинция (размер: 10x10 км)
│           └── Участок (размер: 1x1 км)
│               └── Локальная карта (строительство)
```

### 4.2 Контроль территории

| Действие | Эффект | Требования |
|----------|--------|------------|
| **Исследовать** | Открывает карту | Разведчик/юнит |
| **Заявить** | Помечает как свою | Нет владельца |
| **Захватить** | Принудительное владение | Военная сила |
| **Колонизировать** | Мирное заселение | Пустая земля |
| **Развивать** | Улучшает инфраструктуру | Ресурсы, рабочие |

### 4.3 Влияние на территории

```
ЗОНА ВЛИЯНИЯ:

[Ваша база] ──50%─── [Нейтральная] ──25%─── [Чужая база]
     │                     │                    │
  Контроль             Спорная             Контроль
  100%                 зона                0%

ФАКТОРЫ ВЛИЯНИЯ:
- Расстояние от базы: -10% за 10 км
- Военное присутствие: +20% за отряд
- Инфраструктура: +5% за дорогу/вышку
- Население: +2% за 10 жителей
- Торговые пути: +10% за активный маршрут
```

## 6. ГЕНЕРАЦИЯ МИРА (ТЕХНИЧЕСКИЕ ДЕТАЛИ)

### 6.1 Алгоритм генерации

```python
# Псевдокод генерации мира

def generate_world(seed, size_km=1000):
    """Генерация процедурного мира"""
    
    # 1. Базовый шум для высот
    height_map = perlin_noise(seed, octaves=6, scale=size_km/10)
    
    # 2. Температура (зависит от широты + шум)
    temperature_map = latitude_temperature(size_km) + noise(seed+1, scale=0.3)
    
    # 3. Осадки (зависит от высоты, близости к воде)
    precipitation_map = moisture_noise(seed+2) + orographic_effect(height_map)
    
    # 4. Биомы из температуры и осадков
    biome_map = classify_biomes(temperature_map, precipitation_map)
    
    # 5. Реки (поток с высот)
    river_map = hydraulic_erosion(height_map, iterations=1000)
    
    # 6. Ресурсы (шум с разной частотой)
    resources = {}
    for resource_type in RESOURCE_TYPES:
        frequency = get_resource_frequency(resource_type)
        resources[resource_type] = poisson_disc_sampling(
            seed + hash(resource_type), 
            density=frequency
        )
    
    # 7. Фракции (равномерное распределение)
    factions = place_factions(biome_map, count=8, min_distance=100)
    
    return WorldMap(
        height=height_map,
        temperature=temperature_map,
        precipitation=precipitation_map,
        biomes=biome_map,
        rivers=river_map,
        resources=resources,
        factions=factions
    )
```

### 6.2 Параметры генерации

| Параметр | Значение | Описание |
|----------|----------|----------|
| **Размер мира** | 1000x1000 км | Базовый размер |
| **Разрешение** | 100 м/пиксель | Детализация |
| **Континентов** | 3-7 | Случайное количество |
| **Горных хребтов** | 5-15 | Зависит от размера |
| **Рек** | 50-200 | Длина > 50 км |
| **Озёр** | 20-100 | Размер > 1 км² |
| **Ресурсных мест** | 500-2000 | Все типы |

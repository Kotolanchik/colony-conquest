## 1. ОБЗОР СИСТЕМЫ

### 1.1 Концепция

Религиозная система добавляет **духовный измерение** в игру:
- Разные религии с уникальными учениями
- Влияние веры на поведение поселенцев
- Конфликты между конфессиями
- Священные войны и крестовые походы

## 4. СИСТЕМА ВЕРЫ

### 4.1 Уровень веры

| Уровень | Вера | Эффекты |
|---------|------|---------|
| **0-20** | Атеист | Нет бонусов, +10% научный прогресс |
| **21-40** | Сомневающийся | +5% мораль |
| **41-60** | Верующий | +10% мораль, +5% стабильность |
| **61-80** | Религиозный | +15% мораль, +10% стабильность, +5% продуктивность |
| **81-100** | Фанатик | +20% мораль, +15% стабильность, +10% продуктивность, риск радикализации |

### 4.2 Факторы влияния на веру

| Фактор | Эффект |
|--------|--------|
| **Посещение храма** | +1/неделя |
| **Молитва** | +0.5/день |
| **Благополучие** | +0.2/день |
| **Несчастье** | -0.5/день (ищут утешение в вере) |
| **Образование** | -0.3/день (наука против религии) |
| **Проповеди** | +1/проповедь |
| **Чудо/пророчество** | +10 (единоразово) |

|---------|----------|----------------------|
| **Крестовый поход** | Освободить святое место | +30% мораль |
| **Джихад** | Священная война против неверных | +25% мораль |
| **Защита веры** | Защита от религиозного гнёта | +20% мораль |
| **Конверсия** | Принудительное обращение | +15% мораль |

### 5.2 Последствия

```
ПОБЕДА В СВЯЩЕННОЙ ВОЙНЕ:
• +50 отношения с сорелигионистами
• +20% вера у населения
• Возможность обращения побеждённых
• Престиж

ПОРАЖЕНИЕ В СВЯЩЕННОЙ ВОЙНЕ:
```

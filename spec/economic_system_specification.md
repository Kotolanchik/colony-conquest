## 2. ПРОИЗВОДСТВЕННЫЕ ЦЕПОЧКИ

### 2.1 Формула эффективности производства

```
БАЗОВАЯ ФОРМУЛА ПРОИЗВОДСТВА:

Фактическая_скорость = Базовая_скорость × Модификатор_рабочих × 
                       Модификатор_энергии × Модификатор_инструментов ×
                       Модификатор_мастерства × (1 - Штраф_износа)

ГДЕ:
- Модификатор_рабочих = 0.5 + (Количество_рабочих / Оптимальное_количество)^0.7
- Модификатор_энергии = 1.0 (при 100% энергии), линейно падает до 0.3
- Модификатор_инструментов = 1.0 → 0.5 (при полном износе)
- Модификатор_мастерства = 1.0 + (Средний_навык / 100) × 0.5
- Штраф_износа = Износ_здания / Максимальный_износ
```

```
                (0.5 + 0.5 × √(Рабочие_факт / Рабочие_оптимум)) ×
                (Энергия_факт / Энергия_требуется)^0.8 ×
                (1 - Износ_фактор) ×
                (1 + Средний_навык_рабочих / 200) ×
                (1 + Бонус_мастера / 100)

ГДЕ:
- Износ_фактор = Текущий_износ / (Макс_износ × 2)
- Бонус_мастера: +10% за мастера на каждые 20 рабочих
- При энергии < 50%: штраф -30% к скорости
- При энергии < 25%: штраф -60% к скорости
```

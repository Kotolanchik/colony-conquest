# Спецификации Colony & Conquest

## Порядок чтения

1. **`COLONY_CONQUEST_MASTER_SPECIFICATION.md`** — видение, системная картина, интеграция подсистем (разделы 3–4), стек в общих чертах. Краткий обзор ключевых доменов — **§3**; полный каталог файлов — **§7.1** (и §3.11 со ссылкой на него).
2. **`technical_architecture_specification.md`** — Unity + DOTS, ECS, сеть, производительность: опора для кода в `game/`.
3. **Подсистемы из мастер-документа (раздел 3)** — в логике зависимостей:
   - симуляция поселенцев → экономика / производство → бой;
   - строительство, технологии, глобальная карта;
   - дипломатия, события/квесты, UI/UX.
4. **Остальные файлы в этом каталоге** — углубление доменов (сельское хозяйство, экология, политика и т.д.). Полный список и первый заголовок: **[`../dev/spec_index.md`](../dev/spec_index.md)** (обновление: `python tools/spec_index.py`).

## Работа с изменениями

После правок в `spec/*.md` запускайте `python tools/spec_index.py`. Крупные архитектурные повороты — ADR в `docs/decisions/` и ссылка из связанной задачи в `dev/tasks/`.

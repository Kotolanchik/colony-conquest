## 3. ЭКОНОМИКА

### 3.1 Макроэкономика

| Показатель | Описание | Расчёт |
|------------|----------|--------|
| **ВВП** | Валовой внутренний продукт | Сумма всего производства |
| **ВВП на душу** | ВВП / население | ВВП / Население |
| **Рост ВВП** | % изменения ВВП | (ВВПтек − ВВПпред) / ВВПпред × 100% |
| **Инфляция** | Рост цен | Средний рост цен |
| **Безработица** | % безработных | Безработные / Рабочая сила × 100% |

### 3.2 Структура экономики

```
СТРУКТУРА ВВП:

Первичный сектор (добыча, сельское хозяйство): ___%
Вторичный сектор (промышленность): _____________%
Третичный сектор (услуги): ____________________%

Визуализация:
┌─────────────────────────────────────────┐
│ Первичный   │████████░░░░░░░░░░░░│ 20% │
│ Вторичный   │████████████████░░░░│ 50% │
│ Третичный   │████████████░░░░░░░░│ 30% │
└─────────────────────────────────────────┘
```

### 3.3 Торговля

| Показатель | Описание |
|------------|----------|
| **Экспорт** | Объём продаж за рубеж |
| **Импорт** | Объём покупок из-за рубежа |
| **Торговый баланс** | Экспорт − Импорт |
| **Основные партнёры** | Топ-5 торговых партнёров |
| **Основные товары** | Топ-5 экспортных/импортных товаров |

## 6. СОЦИАЛЬНЫЕ ПОКАЗАТЕЛИ

### 6.1 Качество жизни

| Показатель | Диапазон | Описание |
|------------|----------|----------|
| **Счастье** | 0-100% | Общее удовлетворение |
| **Здоровье** | 0-100% | Состояние здоровья |
| **Образование** | 0-100% | Грамотность |
| **Безопасность** | 0-100% | Уровень преступности |
| **Экология** | 0-100% | Качество окружающей среды |

### 6.2 Индекс человеческого развития (ИЧР)

```
ИЧР = (Образование + Здоровье + Доход) / 3

КАТЕГОРИИ:
• 0.0-0.5: Низкое развитие
• 0.5-0.7: Среднее развитие
• 0.7-0.9: Высокое развитие
• 0.9-1.0: Очень высокое развитие
```

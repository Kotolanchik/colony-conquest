## 6. ИНТЕГРАЦИЯ С ДРУГИМИ СИСТЕМАМИ

| Система | Связь |
|---------|-------|
| **Экономика** | Заводы потребляют ресурсы, производят товары |
| **Военная** | Военные заводы производят оружие и технику |
| **Технологии** | Новые эпохи открывают новые заводы |
| **Поселенцы** | Рабочие нужны для производства |
| **Энергетика** | Заводы требуют электричество |
| **Торговля** | Избыток/дефицит → торговля |

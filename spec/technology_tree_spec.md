## 1. ОБЗОР СИСТЕМЫ

### 1.1 Концепция

Дерево технологий в Colony & Conquest охватывает **5 эпох развития** от XVI века до далёкого будущего. Каждая эпоха содержит уникальные технологии, открывающие новые возможности для строительства, производства, войны и общества.

## 2. СИСТЕМА ИССЛЕДОВАНИЙ

### 2.1 Очки исследований

```
ИСТОЧНИКИ ОЧКОВ ИССЛЕДОВАНИЙ:

Базовое производство:
- Исследовательский стол (Эпоха 1): 10 очков/день
- Библиотека (Эпоха 2): 50 очков/день
- Лаборатория (Эпоха 3): 200 очков/день
- Научный институт (Эпоха 4): 1000 очков/день
- Суперкомпьютер (Эпоха 5): 5000 очков/день

Бонусы:
- Учёный поселенец: +20% за уровень навыка
- Книги/знания: +10-50%
- Внешняя торговля идей: +20-100%
- Образование населения: +5% за 10% грамотности
```

### 2.2 Формула исследования

```
ОЧКИ_В_ДЕНЬ = Базовое_производство × 
              (1 + Навык_учёных × 0.2) ×
              (1 + Бонусы_инфраструктуры) ×
              (1 + Бонусы_технологий) ×
              Модификатор_эпохи

МОДИФИКАТОР_ЭПОХИ:
- Эпоха 1: 1.0x
- Эпоха 2: 1.5x
- Эпоха 3: 2.0x
- Эпоха 4: 3.0x
- Эпоха 5: 5.0x
```

### 2.3 Требования к исследованию

| Тип требования | Описание | Пример |
|----------------|----------|--------|
| **Очки** | Нужно накопить N очков | 1000 очков |
| **Предшественники** | Нужно изучить другие технологии | "Сталь" требует "Плавку железа" |
| **Ресурсы** | Нужно иметь ресурсы | 100 стали |
| **Здания** | Нужно построить | Лаборатория |
| **Население** | Минимум поселенцев | 50 человек |
| **Эпоха** | Достигнуть эпохи | Эпоха 3 |

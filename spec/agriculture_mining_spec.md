### 1.2 Система выращивания

```
ЦИКЛ ВЫРАШИВАНИЯ:

[Подготовка] → [Посев] → [Рост] → [Уход] → [Урожай] → [Хранение]
     │            │          │         │          │           │
   1 день      1 час     5-30 дней  ежедневно   1 день    до 1 года

ФАКТОРЫ УРОЖАЙНОСТИ:
• Плодородие почвы: 0.5x (бедная) до 2.0x (чернозём)
• Вода: -50% без полива
• Удобрения: +20-100%
• Навык фермера: +5% за уровень
• Погода: +20% (идеально), -30% (засуха/наводнение)
• Сезон: +10% в подходящий сезон, -50% в неподходящий
• Вредители/болезни: -10..-50%
```

### 1.3 Формула урожая

```
УРОЖАЙ = Базовый_урожай × 
         Плодородие_почвы ×
         (1 + Удобрения) ×
         (1 + Навык_фермера × 0.05) ×
         Модификатор_погоды ×
         Сезонный_модификатор ×
         (1 - Вредители) ×
         Водный_модификатор

ГДЕ:
- Водный_модификатор: 0.5 (нет воды), 1.0 (есть), 1.2 (автополив)
```

### 2.2 Способы добычи

#### Ручная добыча (Эпоха 1)

```
РУЧНАЯ ДОБЫЧА:

Инструменты:
• Каменная кирка: 1 ед/час, износ 20 ед
• Медная кирка: 2 ед/час, износ 50 ед
• Бронзовая кирка: 3 ед/час, износ 100 ед
• Железная кирка: 5 ед/час, износ 200 ед
• Стальная кирка: 8 ед/час, износ 500 ед

Навык шахтёра: +10% за уровень
Усталость: -1% эффективности за час работы
```

### 2.3 Исчерпаемость месторождений

```
ИСЧЕРПАЕМОСТЬ:

Месторождение имеет запас: 1000 - 1000000 единиц

КАЧЕСТВО РУДЫ:
• Богатая: 80-100% содержание (первые 20% запаса)
• Средняя: 50-80% содержание (следующие 50%)
• Бедная: 20-50% содержание (последние 30%)

ЭФФЕКТ ИСЧЕРПАНИЯ:
• 75% запаса: -10% производства
• 50% запаса: -25% производства
• 25% запаса: -50% производства
• 10% запаса: -75% производства
• 0%: Шахта закрывается

ВОЗОБНОВЛЕНИЕ:
• Дерево: 1% в год (если лесовосстановление)
• Дикие животные: 5% в год
• Рыба: 10% в год (при устойчивом вылове)
```

## 3. ИНТЕГРАЦИЯ С ДРУГИМИ СИСТЕМАМИ

| Система | Связь |
|---------|-------|
| **Экономика** | Сельхозпродукты → переработка → рынок |
| **Поселенцы** | Фермеры, шахтёры — профессии с навыками |
| **Технологии** | Новые эпохи открывают новые методы |
| **Экология** | Удобрения, добыча влияют на окружающую среду |
| **Торговля** | Избыток/дефицит ресурсов → торговля |

# ЗВУКОВОЙ ДИЗАЙН — ТЕХНИЧЕСКАЯ СПЕЦИФИКАЦИЯ
## Colony & Conquest — Аудио система

---

## 1. ОБЗОР СИСТЕМЫ

### 1.1 Концепция

Звуковой дизайн в Colony & Conquest создаёт:
- **Атмосферу** каждой эпохи
- **Обратную связь** для игрока
- **Эмоциональное воздействие**
- **Информацию** о происходящем

### 1.2 Технологии

| Компонент | Технология | Описание |
|-----------|------------|----------|
| **Аудио движок** | FMOD / Wwise | Профессиональное аудио middleware |
| **3D звук** | HRTF / Spatial Audio | Позиционирование в пространстве |
| **Адаптивная музыка** | Horizontal/Vertical Resequencing | Музыка реагирует на игру |
| **Процедурный звук** | Synthesis | Генерация звуков на лету |

## 2. МУЗЫКА

### 2.1 Музыкальные темы по эпохам

#### Эпоха 1: Основание (XVI-XVII век)

```
СТИЛЬ: Народная, акустическая

ИНСТРУМЕНТЫ:
• Лютня/Гитара
• Флейта
• Барабан
• Скрипка

НАСТРОЕНИЕ:
• Спокойное — исследование, строительство
• Напряжённое — опасность, ночь
• Торжественное — победа, праздник
• Грустное — потери, зима
```

#### Эпоха 2: Индустриализация (XIX век)

```
СТИЛЬ: Классическая, романтическая

ИНСТРУМЕНТЫ:
• Струнный оркестр
• Фортепиано
• Духовые
• Перкуссия

НАСТРОЕНИЕ:
• Оптимистичное — прогресс
• Драматичное — социальные конфликты
• Эпическое — промышленный подъём
```

#### Эпоха 3: Первая мировая (1900-1918)

```
СТИЛЬ: Мрачная, военная

ИНСТРУМЕНТЫ:
• Маршевые барабаны
• Медные духовые
• Струнные (тремоло)
• Орган

НАСТРОЕНИЕ:
• Тревожное — перед боем
• Героическое — атака
• Трагическое — потери
• Отчаянное — окружение
```

#### Эпоха 4: Вторая мировая (1918-1945)

```
СТИЛЬ: Эпическая, кинематографичная

ИНСТРУМЕНТЫ:
• Полный оркестр
• Хор
• Электрогитара (редко)
• Синтезаторы (конец эпохи)

НАСТРОЕНИЕ:
• Напряжённое — разведка
• Мощное — танковая атака
• Трагическое — бомбардировки
• Торжественное — победа
```

#### Эпоха 5: Современность/Будущее (1945+)

```
СТИЛЬ: Электронная, экспериментальная

ИНСТРУМЕНТЫ:
• Синтезаторы
• Электронные ударные
• Сэмплы
• Оркестр (гибрид)

НАСТРОЕНИЕ:
• Футуристическое — исследования
• Киберпанковское — города
• Космическое — космос
• Тревожное — кибератаки
```

### 3.3 3D позиционирование

```
ПАРАМЕТРЫ 3D ЗВУКА:

• Расстояние: громкость уменьшается с расстоянием
  └─► Минимальное: 1 м
  └─► Максимальное: 1000 м

• Азимут: лево/право
  └─► HRTF для наушников
  └─► Стерео для колонок

• Элевация: верх/низ
  └─► Фильтрация для высоты

• Окклюзия: препятствия
  └─► Стены ослабляют звук
  └─► Двери/окна пропускают

• Реверберация: отражения
  └─► Комната: 0.5 сек
  └─► Зал: 2 сек
  └─► Открытое пространство: 0 сек
```

## 5. ГОЛОСА

### 5.1 Типы речи

| Тип | Описание | Использование |
|-----|----------|---------------|
| **Баркеры** | Короткие фразы | Бой, действия |
| **Радиопереговоры** | Военная терминология | Командование |
| **Диалоги** | Полные реплики | Сюжет, события |
| **Эмоции** | Вздохи, крики, смех | Реакции |

### 5.2 Языки

```
ПОДХОД К ЯЗЫКАМ:

• Каждая фракция имеет свой "язык"
• Для игрока — перевод на его язык
• Звучание отличается (акцент, интонация)
• Можно изучать языки (навык)

ПРИМЕРЫ:
• Фракция A: германское звучание
• Фракция B: романское звучание
• Фракция C: славянское звучание
```

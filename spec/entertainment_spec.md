## 6. ЭФФЕКТЫ РАЗВЛЕЧЕНИЙ

```
ФОРМУЛА НАСТРОЕНИЯ:

Базовое_настроение + 
(Разнообразие_развлечений × 0.1) + 
(Качество_развлечений × 0.2) + 
(Доступность × 0.15) = Итоговое_настроение

ЭФФЕКТЫ:
• Настроение > 80%: +10% продуктивность
• Настроение 60-80%: +5% продуктивность
• Настроение 40-60%: Базовый уровень
• Настроение 20-40%: -5% продуктивность
• Настроение < 20%: -15% продуктивность, риск бунта
```

## 7. ИНТЕГРАЦИЯ С ДРУГИМИ СИСТЕМАМИ

| Система | Связь |
|---------|-------|
| **Поселенцы** | Развлечения влияют на настроение |
| **Преступность** | Доступность развлечений снижает преступность |
| **Экономика** | Развлекательная индустрия — источник дохода |
| **Строительство** | Спортивные сооружения, театры, бары |
| **Культура** | Искусство формирует культурную идентичность |

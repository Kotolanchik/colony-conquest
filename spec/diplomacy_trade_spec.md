# ДИПЛОМАТИЯ И ТОРГОВЛЯ — ТЕХНИЧЕСКАЯ СПЕЦИФИКАЦИЯ
## Colony & Conquest — Межфракционные отношения

---

## 1. ОБЗОР СИСТЕМЫ

### 1.1 Концепция

Дипломатическая система в Colony & Conquest позволяет:
- Устанавливать **отношения** с другими фракциями (игроками и ИИ)
- Заключать **торговые сделки** и **союзы**
- Вести **шпионаж** и **саботаж**
- Объявлять **войны** и заключать **мир**
- Создавать **федерации** и **империи**

### 1.2 Типы фракций

| Тип | Описание | Особенности |
|-----|----------|-------------|
| **Игрок** | Человек-игрок | Полный контроль |
| **ИИ Фракция** | Компьютерные противники | Разные личности |
| **Нейтральные** | Торговцы, племена | Можно торговать |
| **Дикие** | Животные, мутанты | Всегда враждебны |
| **Древние** | Руины, автоматика | Непредсказуемы |

## 2. СИСТЕМА ОТНОШЕНИЙ

### 2.1 Шкала отношений

```
ШКАЛА ОТНОШЕНИЙ (-100 до +100):

-100 ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ +100
  │                                                                  │
ВРАГИ    НЕДРУЗЬЯ   НЕЙТРАЛ   ДРУЗЬЯ   СОЮЗНИКИ   ВАССАЛ/СЮЗЕРЕН
  │         │         │         │         │           │
 -100      -50        0        +50       +75        +90

УРОВНИ:
• -100..-75: ЯРОСТНЫЕ ВРАГИ — постоянная война
• -75..-50: ВРАГИ — враждебность, рейды
• -50..-25: НЕДРУЗЬЯ — напряжение, инциденты
• -25..+25: НЕЙТРАЛ — нейтралитет
• +25..+50: ДРУЗЬЯ — дружба, торговля
• +50..+75: ХОРОШИЕ ДРУЗЬЯ — союз, помощь
• +75..+90: СОЮЗНИКИ — военный союз
• +90..+100: ВАССАЛ/СЮЗЕРЕН — интеграция
```

| Фактор | Эффект | Период |
|--------|--------|--------|
| **Торговля** | +1 за сделку | Мгновенно |
| **Подарок** | +5..+20 | Мгновенно |
| **Помощь в войне** | +10..+30 | Мгновенно |
| **Общий враг** | +0.5/день | Постоянно |
| **Похожие идеологии** | +0.2/день | Постоянно |
| **Разные идеологии** | -0.2/день | Постоянно |
| **Граничащие территории** | -0.1/день | Постоянно |
| **Оскорбление** | -5..-15 | Мгновенно |
| **Атака** | -20..-50 | Мгновенно |
| **Убийство поселенца** | -10 за каждого | Мгновенно |
| **Нарушение границ** | -2/инцидент | Мгновенно |
| **Шпионаж (раскрыт)** | -15..-30 | Мгновенно |

|-----------|----------|---------------|
| **Милитаризм** | Сила через войну | Конфликт с пацифизмом |
| **Пацифизм** | Мирное развитие | Конфликт с милитаризмом |
| **Капитализм** | Свободная торговля | Синергия с капитализмом |

| **Изоляционизм** | Закрытость от мира | -50% к торговле |

### 3.2 Торговля

#### Типы торговых сделок

| Тип | Описание | Комиссия |
|-----|----------|----------|
| **Мгновенная** | Разовый обмен | 10% |
| **Контракт** | Поставки N дней | 5% |
| **Долгосрочный** | Постоянные поставки | 2% |
| **Бартер** | Обмен без денег | 0% |

#### Торговые товары

| Категория | Примеры | Ценность |
|-----------|---------|----------|
| **Сырьё** | Руда, дерево, камень | Низкая |
| **Материалы** | Сталь, пластик | Средняя |
| **Потребительские** | Еда, одежда | Средняя |
| **Военные** | Оружие, боеприпасы | Высокая |
| **Технологии** | Чертежи, знания | Очень высокая |
| **Рабочая сила** | Рабы, наёмники | Спорная |
| **Информация** | Разведданные | Высокая |

#### Формула цены

```
ЦЕНА = Базовая_цена × 
       (1 + Спрос/Предложение) ×
       (1 + Расстояние_штраф) ×
       (1 + Отношения_бонус) ×
       (1 + Навык_торговца)

ГДЕ:
- Спрос/Предложение: 0.5 (избыток) до 2.0 (дефицит)
- Расстояние_штраф: +10% за 100 км
- Отношения_бонус: -10% (друзья) до +30% (враги)
- Навык_торговца: -5% за уровень
```

### 3.3 Союзы

| Тип союза | Требования | Эффекты |
|-----------|------------|---------|
| **Торговый** | Отношения +25 | -20% комиссия, +50% торговля |
| **Нападения** | Отношения +50 | Авто-война если атакован |
| **Оборонительный** | Отношения +50 | Помощь при обороне |
| **Полный** | Отношения +75 | Всё выше + совместные операции |
| **Федерация** | Отношения +90 | Общее правительство |

### 4.2 Система шпионов

```
АГЕНТЫ:

Уровни агентов:
• Новичок: Базовый шанс успеха
• Опытный: +20% шанс
• Профессионал: +40% шанс
• Мастер: +60% шанс, может обучать

Навыки агентов:
• Скрытность: Шанс остаться незамеченным
• Взлом: Кража технологий
• Саботаж: Разрушение инфраструктуры
• Манипуляция: Дезинформация
• Ликвидация: Убийства

Риск провала = Базовый_риск × (1 - Навык_агента/100)
```

| Условие | Описание | Эффект |

## 9. AI ПОВЕДЕНИЕ

### 9.1 Типы личностей ИИ

| Тип | Характеристики | Поведение |
|-----|----------------|-----------|
| **Агрессор** | Военный фокус | Часто объявляет войны |
| **Торговец** | Экономический фокус | Ищет выгодные сделки |
| **Дипломат** | Мирный фокус | Союзы, федерации |
| **Изоляционист** | Закрытый | Избегает контактов |
| **Империалист** | Экспансия | Захват территорий |
| **Технократ** | Наука | Обмен технологиями |

### 9.2 Принятие решений ИИ

```
РЕШЕНИЕ ИИ О ВОЙНЕ:

Шанс_войны = Базовый_шанс ×
             Модификатор_силы ×
             Модификатор_отношений ×
             Модификатор_личности

ГДЕ:
- Базовый_шанс: 5%/год
- Модификатор_силы: 2.0 (если сильнее), 0.5 (если слабее)
- Модификатор_отношений: 0.1 (друзья), 3.0 (враги)
- Модификатор_личности: 0.5 (пацифист), 2.0 (агрессор)

ЕСЛИ Шанс_войны > random(0, 1):
    Объявить войну
```

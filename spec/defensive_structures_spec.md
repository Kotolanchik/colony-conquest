# ОБОРОНИТЕЛЬНЫЕ СООРУЖЕНИЯ — ТЕХНИЧЕСКАЯ СПЕЦИФИКАЦИЯ
## Colony & Conquest — Полевая фортификация

---

## 1. ОБЗОР СИСТЕМЫ

### 1.1 Концепция

При нападении врага игрок может быстро строить **полевые укрепления** для защиты своих войск. Система охватывает все эпохи — от окопов с лопатами до энергетических щитов.

### 1.2 Типы сооружений

```
КЛАССИФИКАЦИЯ ОБОРОНИТЕЛЬНЫХ СООРУЖЕНИЙ:

┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│  ПОЛЕВЫЕ (строятся во время боя):                                          │
│  ├── Окопы и траншеи                                                        │
│  ├── Мешки с песком                                                         │
│  ├── Колючая проволока                                                      │
│  ├── Противотанковые ежи                                                    │
│  └── Минные поля                                                            │
│                                                                             │
│  УКРЕПЛЁННЫЕ (требуют времени и ресурсов):                                  │
│  ├── ДОТы (долговременные огневые точки)                                    │
│  ├── Бункеры                                                                │
│  ├── Бетонные укрепления                                                    │
│  └── Укреплённые позиции                                                    │
│                                                                             │
│  ЭПОХИ 5 (высокие технологии):                                              │
│  ├── Энергетические щиты                                                    │
│  ├── Силовые поля                                                           │
│  └── Автоматизированные турели                                              │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 5. МЕХАНИКИ УКРЕПЛЕНИЙ

### 5.1 Строительство в бою

```
СКОРОСТЬ СТРОИТЕЛЬСТВА В БОЮ:

Базовая_скорость × 
(1 + Навык_инженера × 0.1) ×
(1 + Количество_рабочих^0.7) ×
Модификатор_под_огнём

МОДИФИКАТОР_ПОД_ОГНЁМ:
• Нет огня: 1.0x
• Дальний огонь: 0.7x
• Ближний огонь: 0.4x
• Штурм: 0.1x (практически невозможно)
```

### 5.2 Разрушение укреплений

| Оружие | Урон сооружениям | Эффективность |
|--------|------------------|---------------|
| **Стрелковое** | 1-5 | Низкая |
| **Пулемёт** | 5-10 | Низкая |
| **Гранатомёт** | 50-100 | Средняя |
| **Артиллерия** | 200-500 | Высокая |
| **Танковое орудие** | 300-800 | Высокая |
| **Авиабомбы** | 500-1500 | Очень высокая |

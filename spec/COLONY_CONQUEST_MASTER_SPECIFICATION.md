## 1. ОБЗОР ПРОЕКТА

### 1.1 Концепция

**Colony & Conquest** — это гибридная игра, сочетающая:
- **Colony Sim**: Глубокая симуляция поселенцев с индивидуальными чертами, навыками, отношениями
- **RTS**: Масштабные сражения с тысячами юнитов
- **Tactical Warfare**: Реалистичная тактика боя с укрытиями, подавлением, логистикой

### 1.2 Ключевая особенность

> **"От лопаты до атомной бомбы"** — плавный переход от выживания одного поселенца к командованию армией в **10,000+ юнитов**.

### 1.3 Вдохновение

| Игра | Что заимствуем |
|------|----------------|
| **RimWorld** | Симуляция поселенцев, психология, социальные связи |
| **Company of Heroes** | Тактика боя, укрытия, подавление |
| **Factorio** | Производственные цепочки, логистика, масштабирование |
| **They Are Billions** | Масштаб, визуальный стиль |
| **Foxhole** | Иерархия командования, логистика армии |

### 1.4 Целевые показатели

| Параметр | Цель |
|----------|------|
| **Масштаб** | 10,000+ юнитов в мире |
| **Производительность** | 60 FPS при 10K юнитов |
| **Камера** | 4 уровня (Micro → Tactical → Operational → Strategic) |
| **Эпохи** | 5 эпох развития (XVI век — будущее) |
| **Мультиплеер** | До 8 игроков, lockstep + server authority |

## 3. СПЕЦИФИКАЦИИ ПОДСИСТЕМ

### 3.1 Система симуляции поселенцев

**Файл:** `settler_simulation_system_spec.md`

**Ключевые компоненты:**
- 40+ черт характера с системой несовместимости
- 20 навыков с потолком таланта (8-18 генетический)
- Психологическая модель: настроение, стресс, ПТСР (0-100)
- 16 типов фобий с триггерами
- Социальные связи: 11 типов отношений (-100 до +100)
- Физиология: 16 частей тела, 9 типов ран
- 5 уровней автономности ИИ

**ECS Components:**
```csharp
// Core
SettlerIdentity, SettlerAppearance, PersonalityTraits, Aptitudes
MentalConditions, SkillSet, PsychologyState, SocialBonds
PhysiologyState, NeedsState, InjuryTracker, MedicalConditions
AutonomyLevel, CurrentTask, AIState, CommandHierarchy
```

### 3.2 Военная система

**Файл:** `military_system_specification.md`

**Ключевые компоненты:**
- 10 уровней иерархии командования (отделение → штаб)
- 9 типов пехоты, 8 типов бронетехники, 6 типов артиллерии
- 8 типов укрытий (20% → 95% защиты)
- Система подавления с 6 уровнями эффектов
- Медицинская система: 5 типов ран, эвакуация, госпитали
- День/ночь, погода, усталость — тактические модификаторы

**Боевые характеристики (пример):**
| Тип | HP | Броня | Скорость | Урон |
|-----|----|-------|----------|------|
| Стрелок | 100 | 5 | 4 м/с | 25 |
| Средний танк | 1500 | 150/100/60/40 | 45 км/ч | 280 |
| Снайпер | 80 | 3 | 3.5 м/с | 150 (×3 голова) |

### 3.3 Экономическая система

**Файл:** `economic_system_specification.md`

**Ключевые компоненты:**
- 50+ ресурсов по 5 эпохам
- Полные производственные цепочки Factorio-style
- 25+ видов транспорта от носильщиков до космических кораблей
- Система "мечи или плуги" — переключение мирное/военное производство
- Снабжение армии: конвои, парашютные сбросы, полевые склады

**Пример цепочки (Танк Эпохи 4):**
```
Железная руда → Доменная печь → Чугун → Конвертер → Сталь → 
→ Бронетанковый завод → Корпус танка + Гусеницы + Двигатель + Орудие → Танк
(60 мин, 60 стали + 25 брони + 1 двигатель)
```

### 3.4 Техническая архитектура

**Файл:** `technical_architecture_specification.md`

**Ключевые решения:**
- **Движок:** Unity + DOTS (Entities 1.x); редактор: Unity 6 — см. §5.1
- **ECS:** Struct-of-Arrays, chunk-based хранение
- **Многопоточность:** Job System + Burst Compiler
- **Оптимизация:** 4 уровня LOD + Meta-Units + Impostors
- **Сеть:** Гибрид Lockstep + Server Authority + Event-Based

**Производительность:**
| Метрика | Без оптимизации | С оптимизацией |
|---------|-----------------|----------------|
| 10K юнитов | 5 FPS | **60 FPS** |
| Draw Calls | 10,000 | **150** |
| Triangles | 50M | **500K** |
| Memory | 2 GB | **400 MB** |

## 4. ИНТЕГРАЦИЯ СИСТЕМ

### 4.1 Точки интеграции

| Система 1 | Система 2 | Точка интеграции |
|-----------|-----------|------------------|
| Поселенцы | Экономика | Рабочие производят, навыки влияют на скорость |
| Поселенцы | Бой | Солдаты = поселенцы с боевыми навыками |
| Поселенцы | Медицина | Ранения влияют на физиологию |
| Экономика | Бой | Производство оружия, снабжение армии |
| Бой | Медицина | Ранения → эвакуация → лечение |
| Технологии | Все | Эпохи открывают новые возможности |

### 4.3 Иерархия ECS Archetypes

```
ENTITY ARCHETYPES (Иерархия наследования)

BASE ENTITY
├── LocalTransform (позиция, поворот, масштаб)
├── HealthData (здоровье, броня)
└── FactionData (фракция, команда)

    SETTLER (Базовый поселенец)
    ├── NeedsData (голод, жажда, энергия, настроение)
    ├── SkillsData[] (навыки, опыт, страсть)
    ├── JobData (текущая работа, прогресс)
    └── PersonalityTraits (черты характера)

        COMBAT_SETTLER (Боевой поселенец)
        ├── CombatData (урон, дальность, кулдаун)
        ├── WeaponData[] (вооружение)
        ├── UnitTypeData (тип: пехота/техника)

    BUILDING (Здание)
    ├── BuildingData (тип, уровень, прогресс стройки)
    ├── PowerGridData (энергопотребление/генерация)
```

### 7.3 Уникальные продажные фичи (USP)

1. **"От лопаты до атомной бомбы"** — единственная игра, где вы видите, как ваш первый поселенец копает грядку, а через 20 часов игры вы командуете ядерным ударом

2. **Живая история** — каждый погибший солдат — это бывший поселенец с именем, историей, семьёй. Игрок видит последствия войны

3. **Реалистичная масштабируемость** — никаких "20 танков = армия". Реальные дивизии, реальные потери, реальная логистика

4. **Эмерджентный геймплей** — истории рождаются из систем, а не скриптов. "Помните сержанта Василия, который в одиночку удержал переправу 3 часа?"
